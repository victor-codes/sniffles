Introduction

CSS is one of the core languages of the open Web and is standardized across Web browsers according to the W3C specification. Previously development of various parts of CSS specification was done synchronously, which allowed versioning of the latest recommendation. You might have heard about CSS1, CSS2.1, CSS3.
Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to red.

p {
color: red;
}

what you should know already

You should learn the basics of HTML before attempting any CSS. It is important that you learn HTML brfore learning CSS.
Once you understand the fundamentals of HTML, we recommend that you learn HTML and CSS at the same time, moving back and forth between the two topics. This is because HTML is far more interesting and much more fun to learn when you apply CSS, and you can't really learn CSS without knowing HTML.

CSS ruleset

Dissecting the CSS code for red paragraph text to understand how it works:

The whole structure is called a ruleset, the names of the individual part are:

Selectors

This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled (in this example, p elements). To style a different element, change the selector.

Declaration

This is a single rule like color: red;. It specifies which of the element's properties you want to style.

properties

These are ways in which you can style an HTML element. (In this example, color is a property of the p elements.) In CSS, you choose which properties you want to affect in the rule.

property value

To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many color values in addition to red.

Other important parts are:

  • Apart from the dicussed part, each ruleset must be wrapped in curly braces. ({})
  • Within each declaration, a colon must be used (:) to separate the property from it's value or values.
  • Within each ruleset, a semicolon must be used (;) to separate each declaration from the next one.

multiple property value can be used in one ruleset, by separating them with a semicolons like this

p { color: red;
width: 500px;
border: 1px solid black;
}

Selecting multiple element

It is possible to select multiple elements and apply one ruleset to all of them. Separate multiple selectors by commas. For example:

p, li, h1 {
color: red;
}

selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Types of selectors

There are many different types of selectors. The examples above use element selectors, which select all elements of a given type. But we can make more specific selections as well. Here are some of the more common types of selectors:

Element selector

It is sometimes called a tag or type selector, It selects all HTML elements of the specified type. For example: p selects p tag

ID selectors

The element on the page with the specified ID. On a given HTML page, each id value should be unique. For example: #my-id selects p id="my-id"> or a id="my-id">

Class selectors

The element(s) on the page with the specified class. Multiple instances of the same class can appear on a page. For example: .my-class selects p class="my-class">.

Attribute selectors

The element(s) on the page with the specified attribute. For example: img[src] selects img src= "myimage.png"> but not img>

Pseudo-class selectors

The specified element(s), but only when in the specified state. (For example, when a cursor hovers over a link.) For example:a:hover selects a tag , but only when the mouse pointer is hovering over the link.

There are many selectors that are not dicussed here.

fonts and texts

Now that we've explored some CSS fundamentals, we can use the font property to improve the styling of an HTML page. Here is an example:

html {
font-size: 10px; /* px means "pixels": the base font size is now 10 pixels high */
font-family: "Open Sans", sans-serif; /* this should be the rest of the output you got from Google fonts */
}

Note: Anything between the /**/ is a CSS comment

Analysizing the property above:
font-family it is used to specify a font(s) to use for a text in an element(s).
font-size to set a particular size for a text.
This rule defines a global base font and font size for the whole page. Since the html tag is the parent element of the whole page, all elements inside it inherit the same font-size and font-family.

CSS box modal

Something you'll notice about writing CSS: a lot of it is about boxes. This includes setting size, color, and position. Most HTML elements on your page can be thought of as boxes sitting on top of other boxes.

CSS layout is mostly based on the box model. Each box taking up space on your page has properties like:

  • padding, the space around the content. In the example below, it is the space around the paragraph text.
  • border, the solid line that is just outside the padding.
  • margin, the space around the outside of the border.

We can also use:

  • width() of an element
  • backgroung-color, the color behind an element's content and padding.
  • color, the color of an element's content (usually text).
  • text-shadow, sets a drop shadow on the text inside an element.
  • display, sets the display mode of an element. (keep reading to learn more).
changing the page color

This rule sets a background color for the entire page. Change the color code to the color you want.

html {
background-color: #00539F;
}

styling the body

A body can be styled by usibg various CSS proerty tag here is an example:

body {
width: 600px;
margin: 0 auto;
background-color: #FF9500;
padding: 0 20px 20px 20px;
border: 5px solid black;
}

Let's Analysize the code above:

  • width: 600px; This forces the body to always be 600 pixels wide.
  • margin: 0 auto;When you set two values on a property like margin or padding, the first value affects the element's top and bottom side (setting it to 0 in this case); the second value affects the left and right side. (Here, auto is a special value that divides the available horizontal space evenly between left and right).
  • background-color: #FF9500; This sets the element's background color. This project uses a reddish orange for the body background color, as opposed to dark blue for the html element.
  • padding: 0 20px 20px 20px;This sets four values for padding. The goal is to put some space around the content. In this example, there is no padding on the top of the body, and 20 pixels on the right, bottom and left. The values set top, right, bottom, left, in that order.
  • border: 5px solid black;This sets values for the width, style and color of the border. In this case, it's a five-pixel–wide, solid black border, on all sides of the body.
Positioning

Positioning allows you to take elements out of the normal document layout flow, and make them behave differently, for example sitting on top of one another, or always remaining in the same place inside the browser viewport.

CSS Flexbox

Flexbox is a one-dimensional layout method for laying out items in rows or columns. Items flex to fill additional space and shrink to fit into smaller spaces.

CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web. It lets you lay content out in rows and columns, and has many features that make building complex layouts straightforward.
